======================     テーブル作成     ======================= 


・テーブル作成

	CREATE TABLE [tablename] (

		

	)



〜データ型〜

	◯文字
		・VARCHAR → 最大サイズ指定可変長文字列
		・CHAR    → 固定長文字列
		・TEXT    → 制限なし可変長文字列

	◯数値
		・INTEGER → 4バイト整数型
		・DECIMAL → 可変精度数値型

	◯日付
		・DATE    → 日付型



〜制約〜

	・NOT NULL

		NOT NULL制約は単一の列に対して定義される。
		該当する列のNULL値を禁止する。


	・UNIQUE (一意性)

		列、または複数の列の組み合わせに対して定義される。
		それらの値が一意であるデータに限りデータの挿入・更新を許可。


	・PRIMARY KEY (主キー)

		列、または複数の列の組み合わせに対して定義され、表の中の行を一意に識別する。
		主キーは1つのテーブルに1つしか設定できない。
		NOT NULL制約も加わる。


	・FOREIGN KEY（外部キー）

		テーブル1とテーブル2の間をテーブル内の列で参照という関連付けを定義する。
		テーブル同士の関連付けを行い、子テーブルは親テーブルを参照し、子テーブルの参照列を外部キーを呼ぶ。
		親テーブルと子テーブルの間で整合性を保つ。
		子テーブルに入力される値は親テーブルに存在する値かNULLでなければならない。


	・CHECK （チェック）

		入力条件を定義する。
		列に条件を満たす値のみ入力が許可される。



〜　sampleテーブルを作成　〜

	CREATE TABLE sample (
	[カラム名] [データ型] [制約],
	[カラム名] [データ型] [制約],
	[カラム名] [データ型] [制約],
	);


	※制約は省略可能
